## During the session (Automatic — no permission needed)
- **Architecture:** When a design decision or tech stack choice is made → `save_note` to `arch` store.
- **Bugs:** When an error is fixed → `save_note` with format `"error -> fix"` to `errors` store.
- **Themes:** When a UI/UX pattern or design token is chosen → `save_note` to `theme` store.
- **Tasks:** When a task is completed or a new goal set → `save_note` to `active_task` store.

## Global Store Guidelines
- If a preference or pattern applies to ALL your projects → `save_note` to `preferences` (global).
- If a library or tool is chosen as a personal standard → `save_note` to `tech_stack` (global).
